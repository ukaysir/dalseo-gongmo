민간주차장 정보제공 신청

  • 대구시 통합주차정보시스템 서비스가 공영주차장 중심에서 민영주차장까지 확대 운영될 예정입니다.
  • 『 대구통합주차정보시스템 』은 대구 전역 주차장을 직접 방문하지 않아도 주차 가능 면수를 실시간으로 확인할 수 있는 서비스입니다.
  • 대구시는 민간주차장 정보 제공을 위해 민간협력을 지속적으로 추진할 계획이며, 민간에서 운영 중인 주차장에 대한 현황 정보나 실시간 연계를 위한 신청 창구를 별도로 마련하였습니다.
  • 민간에서 운영 중인 실시간 주차정보 또는 주차장 정보를 대구 통합주차정보시스템에 제공하여 시민들이 편리하게 이용할 수 있도록 정보 연계를 희망하는 경우 화면의 동의 절차에 따라 신청하여 주시기 바랍니다.

이용 절차

  • 실시간 주차면수 제공 여부에 따라 절차가 달라집니다. 아래에서 선택하세요.
  1. 정보제공 신청 신청서의 모든 항목 입력
    → 실시간 주차면수 제공여부: 미제공 선택
  2. 신청하기 입력 내용 확인 후 신청을 완료합니다.
  3. 담당자 확인 및 접수 담당자가 신청서를 검토하고 접수합니다.
  4. 승인 또는 거절 처리 상태는 마이페이지에서 확인 가능합니다.
  5. 서비스 반영 승인 시 해당 주차장 정보가 대구 통합주차정보시스템에 제공됩니다.
  1. 정보제공 신청 신청서의 모든 항목 입력
    → 실시간 주차면수 제공여부: 제공 선택
  2. 신청하기 입력 내용 확인 후 신청을 완료합니다.
  3. 시 담당자 접수 및 승인/거절 담당자가 신청서를 검토하고 승인/거절합니다.
    처리 상태는 마이페이지에서 확인 가능합니다.
  4. API 키 발급 승인 시 신청자 이메일로 API 키가 발송됩니다.
  5. API 이용 발급된 키로 실시간 주차면수 정보 연계 서비스 및 주차장정보 조회 서비스를 호출 할 수 있습니다. (API 문서 참고)
  6. 서비스 반영 연계된 실시간 정보가 주차정보시스템에 제공됩니다.

실시간 주차면수 정보 연계 서비스

  • 주차장의 실시간 데이터를 API 형태로 제공하여 통합주차관리시스템과 연계합니다.
  • 주차장의 혼잡 상태, 잔여 주차구획 수, 전용주차구역 현황 등의 정보를 실시간으로 수집하여 시민들에게 제공합니다.
  • 신청한 주차장의 아이디와 API key를 이용하여 연계가 가능합니다.(마이페이지 - 민간주차장 신청 상세내역에서 확인가능)
  • 실시간 주차면수 정보 연계 API 상세정보는 실시간 주차면수 정보 연계 API 상세.pdf 를 참고하시기 바랍니다.

주차장 정보 조회 서비스

  • 대구시 내 공영 및 민간주차장의 상세 정보를 조회할 수 있는 서비스입니다.
  • API를 통해 주차장 위치, 운영정보, 요금정보, 전용주차구역 현황 등을 확인할 수 있습니다.
  • 신청한 주차장의 아이디와 API key를 이용하여 API 호출이 가능합니다.(마이페이지 - 민간주차장 신청 상세내역에서 확인가능)
  • 주차장정보 조회 API 상세정보는 주차장정보 조회 API 상세.pdf 를 참고하시기 바랍니다.
