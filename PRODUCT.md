# Product

## Register

product

## Users

달서구 주민, 생활권 사업자, 공모전 심사자가 주요 사용자다. 사용자는 특정 주소나 생활권 주변에서 진행 중인 행정, 교통, 안전, 복지, 환경 이슈가 실제 생활에 어떤 영향을 주는지 빠르게 판단하려고 들어온다.

## Product Purpose

달서 AI 내 주소 영향 요약은 달서구 공식 웹, Supabase 저장 데이터, 대구 공공 API 출처를 주소 기준으로 묶어 지도, 쉬운말 요약, 근거 링크, 주민 행동 가이드로 보여주는 공공서비스 MVP다. 성공 기준은 사용자가 주소를 입력한 뒤 관련 이슈, 거리, 영향 유형, 담당 부서, 원문 확인 경로를 한 화면에서 이해하는 것이다.

## Brand Personality

실용적, 신뢰감 있는, 주민 친화적인 톤을 사용한다. 설명보다 판단을 돕는 화면을 우선하며, 공공기관 파일과 게시판을 그대로 노출하기보다 생활영향 언어로 번역한다.

## Anti-references

마케팅 랜딩처럼 긴 가치 제안을 먼저 보여주는 구조를 피한다. 생성형 AI가 실제로 호출되는 것처럼 과장하지 않는다. 장식적 카드 격자, 과한 그라데이션, 유리 효과, 불필요한 애니메이션, 출처가 약한 요약 표현을 피한다.

## Design Principles

1. 주소 검색과 결과 판단을 첫 화면의 중심에 둔다.
2. 지도, 거리, 영향권을 항상 근거와 함께 보여준다.
3. 요약과 생활영향 코치는 쉬운 한국어로 쓰되 최종 판단은 원문과 담당 부서 확인으로 연결한다.
4. 데이터 상태, 수집 시각, 출처 유형을 숨기지 않는다.
5. 공공서비스답게 익숙한 조작 방식과 높은 가독성을 우선한다.
6. 목록 제목은 `원문 수집`, `영향권` 같은 내부 데이터 표현이 아니라 `주차 혼잡 가능`, `이동 지연 가능`, `보행 안전 확인 필요`처럼 주민이 바로 판단할 수 있는 생활 변화로 쓴다.

## Accessibility & Inclusion

기본 목표는 WCAG AA 수준의 색 대비와 키보드 조작 가능성이다. 고령층과 모바일 사용자를 고려해 작은 회색 글자를 남용하지 않고, 애니메이션은 상태 전환 수준으로 제한한다. 색상만으로 긴급도나 카테고리를 구분하지 않고 텍스트 라벨을 함께 제공한다.
